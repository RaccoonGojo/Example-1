# Тест-кейсы для веб-приложения

## TC-01: Авторизация с корректными данными
**Предусловия:** Пользователь зарегистрирован.  
**Шаги:**  
1. Ввести корректный логин  
2. Ввести корректный пароль  
3. Нажать "Войти"

**Ожидаемый результат:**  
Пользователь успешно авторизован.

---

## TC-02: Добавление товара в корзину
**Шаги:**  
1. Открыть страницу товара  
2. Нажать "Добавить в корзину"

**Ожидаемый результат:**  
Товар появляется в корзине.
