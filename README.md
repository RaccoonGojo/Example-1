# Портфолио тестировщика (QAM)

Всем привет, это мое портфолио тестировщика.  
Здесь представлены примеры моих работ по тестированию веб-приложений и API.  
Репозиторий содержит тестовую документацию, артефакты, примеры баг-репортов и Postman-коллекцию.

---

## Содержание

### 1. Web Testing
Тестирование веб-приложения: функциональное тестирование, тест-дизайн, чек-листы, тест-кейсы и найденные дефекты.  
→ `./web-testing`

---

### 2. API Testing
Проверка API с помощью Postman: позитивные и негативные сценарии, валидация ответов, анализ ошибок.  
→ `./api-testing`

---

## Инструменты
- Postman  
- Jira (пример багов оформлен по шаблону)  
- Chrome DevTools  
- Git, GitHub  
- SQL (базовый уровень)

---

## О репозитории
Этот репозиторий демонстрирует мои навыки тестирования на учебных проектах.  
Работы выполнены самостоятельно, с использованием общепринятых принципов тест-дизайна и оформления QA-документации.
