# Тест-кейсы API (авторизация)

## TC-API-01: Успешная авторизация
**Запрос:** POST /auth  
**Тело:** {"login": "user", "password": "12345"}  

**Ожидаемый результат:**  
- Код 200  
- В ответе есть токен

---

## TC-API-02: Ошибка при пустом пароле
**Запрос:** POST /auth  
**Тело:** {"login": "user", "password": ""}  

**Ожидаемый результат:**  
- Код 400  
- Сообщение об ошибке
